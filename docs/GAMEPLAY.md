## RTS 모드

### 게임 개요

**막아라! 무너트려라!**는 자원을 수집하고, 유닛을 생산하여 적의 기지를 파괴하는 실시간 전략(RTS) 게임입니다.

### 기본 정보

| 항목 | 값 |
|------|-----|
| 게임 시간 | 10분 (600초) |
| 맵 크기 | 4000 x 2400 픽셀 |
| 플레이어 기지 위치 | 좌측 (200, 1200) |
| 적 기지 위치 | 우측 (3800, 1200) |
| 플레이어 기지 HP | 1000 |
| 적 기지 HP | 1000~1500 (난이도별) |

### 난이도 선택

게임 시작 시 AI 난이도를 선택할 수 있습니다.

| 난이도 | 적 기지 HP | AI 행동 주기 | 대량 발생 | 추천 대상 |
|--------|-----------|-------------|----------|----------|
| 쉬움 | 1000 | 5초 | ❌ | 초보자 |
| 중간 | 1200 | 2.5초 | 2분에 1회 | 일반 |
| 어려움 | 1500 | 2초 | 1분30초마다 반복 | 숙련자 |

> 어려움에서는 AI가 한 번에 최대 3유닛을 동시 소환할 수 있습니다.

### 승리 조건

#### 승리

다음 중 하나를 달성하면 **승리**합니다:

1. **적 기지 파괴** - 적 기지의 HP를 0으로 만듦
2. **시간 종료 시 HP 우위** - 10분이 지났을 때 내 기지의 HP가 적보다 높음

#### 패배

다음 중 하나가 발생하면 **패배**합니다:

1. **내 기지 파괴** - 내 기지의 HP가 0이 됨
2. **시간 종료 시 HP 열세** - 10분이 지났을 때 적 기지의 HP가 같거나 높음

### 게임 흐름

#### 1단계: 초반 (0~3분)

- **목표**: 자원 수급 체계 구축
- **권장 행동**:
  - 나무꾼 2~3명 고용
  - 광부 1~2명 고용 (나무 확보 후)
  - 검병으로 기본 방어선 구축

#### 2단계: 중반 (3~7분)

- **목표**: 전력 증강 및 공방 균형
- **권장 행동**:
  - 궁수 생산 시작 (나무 확보됨)
  - 벽 건설로 방어력 강화
  - 지속적인 유닛 생산

#### 3단계: 후반 (7~10분)

- **목표**: 적 기지 공략 또는 HP 우위 확보
- **권장 행동**:
  - 대규모 공격 부대 편성
  - 기지 업그레이드로 HP 확보
  - 필요시 수비 강화

### 조작법

#### 키보드

| 키 | 동작 | 설명 |
|----|------|------|
| `↑` | 카메라 위로 | 맵 상단으로 시점 이동 |
| `↓` | 카메라 아래로 | 맵 하단으로 시점 이동 |
| `←` | 카메라 왼쪽으로 | 맵 좌측으로 시점 이동 |
| `→` | 카메라 오른쪽으로 | 맵 우측으로 시점 이동 |
| `Space` | 본진으로 이동 | 플레이어 기지로 카메라 즉시 이동 |
| `Q` | 벽 배치 모드 | 벽 배치 모드 토글 |
| `W` | 본진 강화 | 본진 업그레이드 |
| `E` | 약초 판매 | 약초 10개를 30골드로 판매 |
| `Y` | 가장자리 스크롤 토글 | 마우스 가장자리 스크롤 켜기/끄기 |
| `ESC` | 일시정지 | AI 대전/튜토리얼에서 일시정지 토글 |
| `M` | 사운드 토글 | 사운드 켜기/끄기 |

#### 마우스

| 동작 | 기능 | 설명 |
|------|------|------|
| 좌클릭 (유닛) | 유닛 선택 | 유닛 정보 확인 |
| 우클릭 드래그 | 카메라 이동 | 드래그 방향으로 시점 이동 |
| 미니맵 클릭 | 해당 위치로 이동 | 클릭한 위치로 카메라 즉시 이동 |
| 마우스 휠 | 줌 인/아웃 | 0.5x ~ 2x 배율 조절 |
| 화면 가장자리 | 카메라 이동 | 가장자리 스크롤 활성화 시 자동 이동 |

> 📷 가장자리 스크롤은 기본적으로 비활성화되어 있습니다. `Y` 키 또는 미니맵 좌측의 📷 버튼으로 토글할 수 있습니다.

#### UI 버튼

| 버튼 | 위치 | 기능 |
|------|------|------|
| 유닛 버튼 | 하단 좌측 | 해당 유닛 생산 |
| 벽 건설 | 하단 중앙 | 방어벽 건설 (HP 150, 30초 후 소멸) |
| 기지 업그레이드 | 하단 중앙 | 기지 HP +200, 골드 수입 +1/초 (1레벨: 150골드, 2레벨+: 복합자원, 최대 5레벨) |
| 사운드 컨트롤 | 미니맵 좌측 | 스피커 아이콘 클릭으로 음소거 토글, 호버 시 볼륨 슬라이더 표시 |

### 전략 팁

#### 초보자 팁

1. **자원 수급이 최우선** - 초반에 지원 유닛(나무꾼, 광부)을 먼저 고용하세요
2. **지원 유닛 활용** - 나무꾼, 광부, 채집꾼을 고용하여 자원을 확보하세요
3. **궁수의 가치** - 궁수는 비싸지만 원거리 공격으로 효율적입니다
4. **벽은 시간을 번다** - 적의 공격을 늦출 수 있습니다 (30초 후 자동 소멸)
5. **힐러로 유닛 보존** - 전투 유닛이 3명 이상이면 힐러를 고용하세요

#### 고급 팁

1. **자원 노드 위치 파악** - 미니맵으로 자원 위치를 확인하세요
2. **적 AI 패턴 파악** - 난이도별로 AI 행동 주기가 다릅니다:
   - 쉬움: 5초마다 60% 확률
   - 중간: 2.5초마다 85% 확률, 2분에 대량 발생 1회
   - 어려움: 2초마다 95% 확률, 1분30초마다 대량 발생 (다중 소환)
3. **적 기지 HP 확인** - 어려움에서는 적 기지 HP가 50% 증가합니다
4. **HP 관리** - 시간 종료 시 HP가 동점이면 패배입니다
5. **AI 자원 관리** - AI도 채집꾼/광부를 고용하고 약초를 판매합니다
6. **수정 확보** - 맵 중앙의 수정(3개)으로 마법사를 생산할 수 있습니다
7. **마법사 활용** - 적이 밀집했을 때 마법사의 범위 공격이 매우 효과적입니다

#### 상황별 대응

| 상황 | 대응 전략 |
|------|----------|
| 자원 부족 | 지원 유닛 추가 고용 |
| 적 러시 | 벽 건설, 검병 대량 생산 |
| 시간 부족 | 공격 유닛 집중 생산, 적 기지 공략 |
| HP 열세 | 기지 업그레이드, 방어 강화 |
| 적 밀집 | 마법사의 범위 공격 활용 |
| 유닛 손실 많음 | 힐러 고용으로 유닛 보존 |

## RPG 모드 (넥서스 디펜스)

### 게임 개요

영웅을 조종하여 중앙의 넥서스를 수호하고, 적 기지를 파괴한 후 보스를 처치하는 협동 게임 모드입니다.

| 항목 | 값 |
|------|-----|
| 맵 크기 | 3000 x 2000 픽셀 |
| 넥서스 HP | 5000 |
| 적 기지 HP | 각 3000 |
| 영웅 부활 시간 | 5초 |

### 승리/패배 조건

| 조건 | 결과 |
|------|------|
| 모든 적 기지 파괴 + 모든 보스 처치 | 승리 |
| 넥서스 HP 0 | 패배 |

### 게임 흐름

#### 1단계: 초반 (0~3분)

- **목표**: 골드 확보 및 업그레이드
- **권장 행동**:
  - 넥서스 근처에서 적 처치
  - 공격력/골드획득률 우선 업그레이드
  - 적이 넥서스에 도달하지 않도록 방어

#### 2단계: 중반 (3~6분)

- **목표**: 적 기지 공략 준비
- **권장 행동**:
  - 충분한 업그레이드 확보
  - 한쪽 기지 집중 공략
  - 넥서스 HP 관리

#### 3단계: 후반 (기지 파괴 후)

- **목표**: 보스 처치
- **권장 행동**:
  - 궁극기 쿨다운 확인 후 보스 도전
  - 보스 어그로 관리 (탱커가 유인)
  - 팀워크로 보스 집중 공격

### 조작법

| 키 | 동작 |
|----|------|
| WASD | 이동 |
| 자동 | 기본 공격 (사거리 내 적 자동 공격) |
| Shift | 스킬 (마우스 방향으로 시전) |
| R | 궁극기 (마우스 방향으로 시전) |
| 1 | 공격력 업그레이드 |
| 2 | 이동속도 업그레이드 |
| 3 | HP 업그레이드 |
| 4 | 공격속도 업그레이드 |
| 5 | 골드획득률 업그레이드 |
| 6 | 사거리 업그레이드 (궁수/마법사) |
| C (유지) | 공격 사거리 표시 |
| Space | 카메라를 영웅 위치로 이동 |
| ESC | 일시정지 |
| 마우스 휠 | 줌 인/아웃 |

### 영웅 클래스별 전략

#### 전사

- **역할**: 근접 딜러, 지속 전투
- **장점**: 피해흡혈로 체력 유지, 광전사 버프로 팀 지원
- **팁**: 광전사 버프 활성화 후 적 밀집 지역에서 전투

#### 궁수

- **역할**: 원거리 딜러, 고데미지
- **장점**: 빠른 공격속도, 높은 공격력, 관통 화살
- **팁**: 거리 유지 필수, 화살 비로 다수의 적 처리

#### 기사

- **역할**: 탱커, 넥서스 수호
- **장점**: 높은 HP, 철벽 방어로 생존, 방패 돌진 기절
- **팁**: 넥서스 근처에서 방어, 철벽 방어로 위기 상황 극복

#### 마법사

- **역할**: 범위 딜러, 기지 공략
- **장점**: 강력한 범위 데미지, 운석 낙하 대범위 공격
- **팁**: 적 밀집 시 화염구/운석으로 처리, 기지 공략에 효과적

### 멀티플레이어 팁

1. **역할 분담**: 탱커(기사)는 넥서스 수호, 딜러는 기지 공략
2. **버프 공유**: 전사 광전사, 기사 철벽 방어는 범위 내 아군에게 적용
3. **기지 분담**: 3~4인 플레이 시 각자 다른 기지 담당
4. **보스전 협력**: 기사가 어그로, 나머지가 딜링
