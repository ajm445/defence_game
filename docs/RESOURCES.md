## 자원 개요

게임에는 5가지 자원이 존재합니다. 각 자원은 고유한 획득 방법과 용도를 가집니다.

### 시작 자원

| 자원 | 초기량 |
|------|--------|
| 골드 | 100 |
| 나무 | 0 |
| 돌 | 0 |
| 약초 | 0 |
| 수정 | 0 |

---

## 자원 종류

### 골드 (Gold)

가장 기본적인 자원입니다. 모든 유닛 생산에 필요합니다.

| 속성 | 값 |
|------|-----|
| 색상 | 금색 |
| 자동 획득 | 기본 4/초 (업그레이드로 증가 가능) |
| 획득 방법 | 자동 획득, 금광부 (광산), 기지 업그레이드 |
| 주요 용도 | 유닛 생산, 업그레이드 |

**특징**:
- 시간이 지나면 자동으로 획득됩니다
- 기본 10분 동안 최대 2,400 골드 자동 획득 가능
- 기지 업그레이드 시 골드 수입 +1/초 증가
- 금광부를 고용하면 광산에서 추가 골드 획득 가능
- 모든 행동의 기본 자원

---

### 나무 (Wood)

건설과 고급 유닛에 필요한 자원입니다.

| 속성 | 값 |
|------|-----|
| 색상 | 갈색 |
| 자동 획득 | 없음 |
| 획득 방법 | 나무꾼 |
| 주요 용도 | 궁수, 광부, 벽 건설 |

**특징**:
- 나무꾼이 자동으로 채집
- 궁수 생산의 핵심 자원
- 초반에 나무꾼 고용이 중요

---

### 돌 (Stone)

건설과 업그레이드, 기사 생산에 필요한 자원입니다.

| 속성 | 값 |
|------|-----|
| 색상 | 회색 |
| 자동 획득 | 없음 |
| 획득 방법 | 광부 |
| 주요 용도 | 기사 생산, 벽 건설, 기지 업그레이드 |

**특징**:
- 광부가 자동으로 채집
- 기사 생산의 핵심 자원 (30 필요)
- 방어 시설에 필수
- 기지 업그레이드에 대량 필요

---

### 약초 (Herb)

골드로 교환할 수 있는 자원입니다.

| 속성 | 값 |
|------|-----|
| 색상 | 초록색 |
| 자동 획득 | 없음 |
| 획득 방법 | 채집꾼 |
| 주요 용도 | 골드 교환 (판매) |

**특징**:
- 채집꾼이 자동으로 채집
- **약초 10개를 골드 30으로 교환 가능**
- 빠른 채집 속도 (1.2/초)

**약초 판매**:
| 필요 약초 | 획득 골드 | 효율 |
|----------|----------|------|
| 10개 | 30 골드 | 3 골드/약초 |

---

### 수정 (Crystal)

희귀한 고급 자원입니다.

| 속성 | 값 |
|------|-----|
| 색상 | 보라색 |
| 자동 획득 | 없음 |
| 획득 방법 | 수정 노드, 채집꾼 (희귀) |
| 주요 용도 | 향후 확장 예정 |

**특징**:
- 맵 중앙에 소수만 존재 (3개)
- 채집꾼이 약초 채집 시 0.1% 확률로 획득
- 가장 희귀한 자원

---

## 자원 획득 방법

### 1. 자동 획득

| 자원 | 획득량 | 조건 |
|------|--------|------|
| 골드 | 4/초 | 없음 (자동, 플레이어/AI 동일) |

```
10분 게임 기준 자동 획득량:
골드 = 4 × 60 × 10 = 2,400
```

### 2. 지원 유닛 채집

지원 유닛이 해당 자원 노드에서 자동으로 채집합니다.

| 유닛 | 채집 자원 | 채집 속도 |
|------|----------|----------|
| 나무꾼 | 나무 | 1.0/초 |
| 광부 | 돌 | 0.8/초 |
| 채집꾼 | 약초 | 1.2/초 |
| 금광부 | 골드 | 1.5/초 |

## 자원 노드

맵에 분포된 자원 노드에서 자원을 획득할 수 있습니다.

### 노드 종류

| 노드 타입 | 제공 자원 | 최대 보유량 | 재생성 시간 |
|----------|----------|------------|------------|
| 나무 (Tree) | 나무 | 100 | 40초 |
| 바위 (Rock) | 돌 | 80 | 90초 |
| 약초 (Herb) | 약초 | 50 | 30초 |
| 수정 (Crystal) | 수정 | 30 | 180초 |
| 광산 (Gold Mine) | 골드 | 80 | 40초 |

### 노드 분포

| 노드 타입 | 개수 | 위치 |
|----------|------|------|
| 나무 | 16개 | 플레이어 측 8개, 적 측 8개 |
| 바위 | 10개 | 맵 중앙 지역 |
| 약초 | 15개 | 맵 전체에 분포 |
| 수정 | 3개 | 맵 정중앙 |
| 광산 | 8개 | 플레이어 측 4개, 적 측 4개 |

### 노드 재생성

자원 노드는 고갈되면 일정 시간 후 재생성됩니다.

```
나무: 40초 후 100으로 재생성
바위: 90초 후 80으로 재생성
약초: 30초 후 50으로 재생성
수정: 180초 후 30으로 재생성
광산: 40초 후 80으로 재생성
```

## 자원 활용

### 유닛 생산 비용

| 유닛 | 골드 | 나무 | 돌 | 약초 | 수정 |
|------|------|------|-----|------|------|
| 검병 | 50 | - | - | - | - |
| 궁수 | 80 | 10 | - | - | - |
| 기사 | 120 | 20 | 30 | - | - |
| 나무꾼 | 30 | - | - | - | - |
| 광부 | 40 | 5 | - | - | - |
| 채집꾼 | 35 | - | - | - | - |
| 금광부 | 100 | 20 | - | - | - |

### 건설/업그레이드 비용

| 행동 | 골드 | 나무 | 돌 | 효과 |
|------|------|------|-----|------|
| 벽 건설 | - | 20 | 10 | HP 200 방어벽 |
| 기지 업그레이드 | 레벨별 | - | 레벨별 | 기지 HP +200, 골드 수입 +1/초 |

#### 기지 업그레이드 비용 (레벨당 1.5배 증가)

| 레벨 | 골드 | 돌 | 누적 골드 수입 |
|------|------|-----|---------------|
| 1 | 100 | 50 | 5/초 |
| 2 | 150 | 75 | 6/초 |
| 3 | 225 | 112 | 7/초 |
| 4 | 337 | 168 | 8/초 |

## 자원 관리 전략

### 초반 (0~3분)

```
목표: 자원 수급 체계 구축

권장 순서:
1. 나무꾼 고용 (30G) - 즉시
2. 나무꾼 추가 (30G) - 30초 후
3. 검병 생산 (50G) - 방어용
4. 광부 고용 (40G + 5W) - 나무 확보 후
```

### 중반 (3~7분)

```
목표: 균형 잡힌 자원 관리

권장 행동:
- 나무꾼 2~3명 유지
- 광부 1~2명 유지
- 궁수 생산 시작
- 벽 1~2개 건설
```

### 후반 (7~10분)

```
목표: 승리 조건 달성

상황별 자원 사용:
- 공격 우선: 궁수/검병에 집중 투자
- 방어 우선: 벽, 기지 업그레이드에 투자
```

### 자원 우선순위

| 순위 | 자원 | 이유 |
|------|------|------|
| 1 | 골드 | 모든 행동의 기본 |
| 2 | 나무 | 궁수, 기사, 건설에 필수 |
| 3 | 돌 | 기사 생산, 방어 시설에 필수 |
| 4 | 약초 | 골드 교환용 (10개 → 30골드) |
| 5 | 수정 | 희귀, 향후 확장용 |
