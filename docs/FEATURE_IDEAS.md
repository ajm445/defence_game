## 신규 유닛

현재 7종 유닛(전투 3종, 지원 4종)에 추가할 수 있는 새로운 유닛들입니다.

### 전투 유닛

#### 힐러 (Healer)

아군 유닛의 HP를 회복시키는 지원형 전투 유닛입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 70 골드 + 15 약초 | 약초 활용 |
| **HP** | 60 | 낮은 체력 |
| **회복량** | 10 HP/초 | 단일 대상 |
| **회복 사거리** | 100 픽셀 | 중거리 |
| **이동 속도** | 1.4 | 보통 |

**특징**:
- 가장 HP가 낮은 아군 유닛을 자동 타겟
- 전투에 직접 참여하지 않음 (공격력 0)
- 약초의 새로운 사용처 제공

---

#### 마법사 (Mage)

범위 공격이 가능한 강력한 원거리 유닛입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 150 골드 + 10 수정 | 수정 활용 |
| **HP** | 40 | 매우 낮은 체력 |
| **공격력** | 35 | 높은 공격력 |
| **범위 피해** | 50 픽셀 반경 | 스플래시 데미지 |
| **공격 사거리** | 180 픽셀 | 장거리 |
| **공격 쿨다운** | 2초 | 느린 공격 |

**특징**:
- 밀집된 적 유닛에 효과적
- 높은 비용과 수정 필요
- 유리대포 스타일 (고위험 고보상)

---

#### 도적 (Rogue)

빠른 이동 속도와 특수 능력을 가진 암살 유닛입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 90 골드 + 15 나무 | 중간 비용 |
| **HP** | 70 | 낮은 체력 |
| **공격력** | 40 | 높은 공격력 |
| **공격 사거리** | 25 픽셀 | 근접 |
| **이동 속도** | 2.5 | 매우 빠름 |
| **특수 능력** | 자원 약탈 | - |

**특징**:
- 적 지원 유닛 사냥에 특화
- 적 지원 유닛 처치 시 해당 유닛이 들고 있던 자원의 50% 획득
- 빠른 이동으로 기습 공격 가능

---

#### 투석기 (Catapult)

건물과 기지에 추가 피해를 주는 공성 유닛입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 200 골드 + 50 나무 + 30 돌 | 높은 비용 |
| **HP** | 150 | 보통 |
| **공격력** | 20 (유닛) / 60 (건물) | 건물 특화 |
| **공격 사거리** | 250 픽셀 | 최장거리 |
| **이동 속도** | 0.8 | 매우 느림 |
| **공격 쿨다운** | 3초 | 매우 느림 |

**특징**:
- 벽과 기지에 3배 피해
- 느린 이동과 공격 속도
- 호위 유닛이 필요함

---

### 지원 유닛

#### 정찰병 (Scout)

시야를 확장하고 적 정보를 제공하는 정찰 유닛입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 40 골드 | 저렴 |
| **HP** | 30 | 매우 낮음 |
| **이동 속도** | 3.0 | 가장 빠름 |
| **시야 범위** | 300 픽셀 | 넓은 시야 |

**특징**:
- 전장의 안개(Fog of War) 시스템과 연계
- 적 유닛 구성 파악 가능
- 전투력 없음

## 건물 시스템

기지 외에 건설 가능한 건물들입니다.

### 방어 타워 (Defense Tower)

자동으로 범위 내 적을 공격하는 방어 건물입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 100 골드 + 30 나무 + 20 돌 | - |
| **HP** | 300 | 벽보다 높음 |
| **공격력** | 15 | 기본 공격력 |
| **공격 사거리** | 200 픽셀 | 넓은 범위 |
| **공격 쿨다운** | 1초 | - |
| **건설 위치** | 본진 800px 이내 | 제한적 |

**특징**:
- 방어 라인 구축에 필수
- 벽과 조합하여 방어 진지 형성
- 최대 3개까지 건설 가능

---

### 막사 (Barracks)

유닛 생산 속도를 증가시키는 건물입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 150 골드 + 50 나무 | - |
| **HP** | 400 | 높은 체력 |
| **효과** | 생산 시간 -20% | - |
| **건설 위치** | 본진 500px 이내 | - |

**특징**:
- 유닛 생산 쿨다운 감소
- 중복 건설 불가 (최대 1개)
- 파괴 시 효과 상실

---

### 대장간 (Blacksmith)

유닛 업그레이드를 연구하는 건물입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 200 골드 + 40 돌 | - |
| **HP** | 350 | 높은 체력 |
| **건설 위치** | 본진 500px 이내 | - |

**업그레이드 목록**:

| 업그레이드 | 비용 | 효과 |
|-----------|------|------|
| 강화 무기 | 100G + 30돌 | 전투 유닛 공격력 +10% |
| 강화 갑옷 | 100G + 30돌 | 전투 유닛 HP +15% |
| 신속 훈련 | 80G + 20나무 | 유닛 이동속도 +10% |
| 효율 채집 | 60G + 20나무 | 지원 유닛 채집속도 +20% |

---

### 창고 (Storage)

자원 보관 한도를 증가시키는 건물입니다.

| 속성 | 제안 값 | 설명 |
|------|---------|------|
| **비용** | 80 골드 + 40 나무 | - |
| **HP** | 200 | 보통 |
| **효과** | 자원 상한 +500 | - |

**특징**:
- 현재 자원 상한이 없다면, 상한 시스템 추가 후 적용
- 여러 개 건설 가능

## 신규 게임 모드

### 서바이벌 모드 (Survival)

끊임없이 밀려오는 적 웨이브를 방어하는 모드입니다.

**규칙**:
- 적 AI 없이 웨이브로 적 유닛 등장
- 웨이브마다 적 강화 (수, 종류, 스탯)
- 기지가 파괴되면 게임 오버
- 생존 웨이브 수로 점수 산정

**웨이브 예시**:

| 웨이브 | 적 구성 | 특징 |
|--------|--------|------|
| 1~5 | 검병 5~15명 | 기본 |
| 6~10 | 검병 + 궁수 | 혼합 |
| 11~15 | 기사 등장 | 탱커 |
| 16+ | 엘리트 유닛 | 강화 스탯 |

---

### 스피드 모드 (Speed)

5분으로 단축된 빠른 경기 모드입니다.

**규칙**:
- 게임 시간: 5분
- 초기 자원 2배
- 골드 수입 2배
- 유닛 생산 속도 1.5배

---

### 킹 오브 더 힐 (King of the Hill)

맵 중앙 거점을 점령하는 모드입니다.

**규칙**:
- 맵 중앙에 거점 존재
- 거점 범위 내 유닛이 많은 쪽이 점령
- 점령 중 1초당 1점 획득
- 100점 먼저 달성 시 승리

---

### 팀 배틀 (Team Battle) - 2v2

2명씩 팀을 이루어 대전하는 모드입니다.

**규칙**:
- 4인 플레이 (2v2)
- 팀원과 자원 공유 가능
- 팀원 기지 중 하나라도 파괴되면 패배
- 확장된 맵 (6000 x 2400)

## UI/UX 개선

### 유닛 그룹 지정

키보드 단축키로 유닛 그룹을 지정하고 선택하는 기능입니다.

**조작**:
- `Ctrl + 1~9`: 현재 선택 유닛을 그룹으로 저장
- `1~9`: 해당 그룹 선택
- `Shift + 1~9`: 해당 그룹을 현재 선택에 추가

---

### 생산 대기열

유닛 생산을 대기열에 추가하는 기능입니다.

**기능**:
- 유닛 버튼 클릭 시 대기열에 추가
- 최대 5개까지 대기 가능
- 대기열 UI 표시
- 우클릭으로 대기열에서 제거 (환불)

---

### 통계 시스템

게임 종료 후 상세 통계를 보여주는 기능입니다.

**표시 항목**:
- 생산한 유닛 수 (종류별)
- 처치한 적 유닛 수
- 채집한 자원량
- 가한/받은 피해량
- 최대 유닛 수
- 건설한 벽/건물 수

---

### 리플레이 시스템

게임을 녹화하고 다시 볼 수 있는 기능입니다.

**기능**:
- 자동 게임 기록
- 재생, 일시정지, 배속 조절
- 특정 시점으로 이동
- 리플레이 저장/불러오기

---

### 튜토리얼 모드

신규 플레이어를 위한 단계별 튜토리얼입니다.

**단계**:
1. 기본 조작 (카메라 이동, 줌)
2. 자원 시스템 설명
3. 유닛 생산 방법
4. 전투 기초
5. 건설 시스템
6. 실전 연습

---

## 기타 아이디어

### 업적 시스템

게임 내 목표를 달성하면 업적을 해금합니다.

| 업적 | 조건 |
|------|------|
| 첫 승리 | 게임에서 처음 승리 |
| 완벽한 방어 | 기지 피해 0으로 승리 |
| 스피드 러너 | 3분 내 승리 |
| 자원왕 | 총 자원 5000 이상 채집 |
| 학살자 | 적 유닛 100기 처치 |

---

### 커스텀 게임 설정

게임 시작 전 세부 설정을 조정할 수 있습니다.

| 설정 | 옵션 |
|------|------|
| 게임 시간 | 5분 / 10분 / 15분 / 무제한 |
| 시작 자원 | 0.5배 / 1배 / 2배 / 5배 |
| 유닛 속도 | 0.5배 / 1배 / 1.5배 / 2배 |
| 기지 HP | 500 / 1000 / 2000 / 5000 |

---

### 랭킹 시스템

멀티플레이어 경쟁 랭킹 시스템입니다.

**구성**:
- ELO 기반 매치메이킹
- 시즌제 랭킹 (브론즈 ~ 다이아몬드)
- 리더보드 표시
- 시즌 종료 보상

---

### 전장의 안개 (Fog of War)

시야 밖 지역이 가려지는 시스템입니다.

**규칙**:
- 유닛/건물 주변만 시야 확보
- 시야 밖 적 유닛은 보이지 않음
- 정찰 유닛으로 시야 확장
- 미니맵에도 적용

## 우선순위 제안

구현 난이도와 게임 개선 효과를 고려한 우선순위입니다.

### 높은 우선순위 (빠른 개선)

| 기능 | 이유 |
|------|------|
| 시각적 효과 | 게임 몰입감 대폭 향상 |
| 수정 활용 | 이미 존재하는 자원 활용 |
| 생산 대기열 | 편의성 크게 향상 |
| 유닛 그룹 지정 | RTS 필수 기능 |

### 중간 우선순위 (핵심 콘텐츠)

| 기능 | 이유 |
|------|------|
| 신규 유닛 (힐러, 투석기) | 전략 다양화 |
| 방어 타워 | 방어 전략 확장 |
| 기지 스킬 | 게임플레이 깊이 추가 |
| 서바이벌 모드 | 새로운 플레이 방식 |

### 낮은 우선순위 (장기 목표)

| 기능 | 이유 |
|------|------|
| 사운드 시스템 | 필수는 아님 |
| 리플레이 시스템 | 구현 복잡도 높음 |
| 랭킹 시스템 | 서버 인프라 필요 |
| 팀 배틀 모드 | 대규모 변경 필요 |
