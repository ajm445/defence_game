## 유닛 개요

유닛은 크게 **전투 유닛**과 **지원 유닛** 두 가지로 분류됩니다.

| 구분 | 역할 | 유닛 종류 |
|------|------|----------|
| 전투 유닛 | 적 유닛/기지 공격 | 검병, 궁수, 기사 |
| 지원 유닛 | 자원 채집 | 나무꾼, 광부, 채집꾼, 금광부 |

### 공통 속성

| 속성 | 설명 |
|------|------|
| HP | 유닛의 체력, 0이 되면 사망 |
| Speed | 이동 속도 (픽셀/프레임) |
| Cost | 생산에 필요한 자원 |

## 전투 유닛

전투 유닛은 적 유닛을 공격하고, 적 기지를 파괴하는 역할을 합니다.

### 검병 (Melee)

근접 전투 전문 유닛입니다. 저렴한 비용으로 대량 생산이 가능합니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 검병 | - |
| **비용** | 50 골드 | 골드만 필요 |
| **HP** | 100 | 높은 체력 |
| **공격력** | 15 | 기본 공격력 |
| **공격 사거리** | 30 픽셀 | 근접 공격 |
| **이동 속도** | 1.5 | 보통 |
| **공격 쿨다운** | 1초 | - |

#### 특징

- **장점**: 저렴함, 높은 체력, 빠른 이동 속도
- **단점**: 근접 공격만 가능, 궁수에게 취약
- **추천 상황**: 초반 러시, 물량 전투, 탱킹

#### 데미지 계산

```
초당 DPS = 공격력 / 쿨다운 = 15 / 1 = 15 DPS
적 기지 파괴 시간 (단독) = 1000 / 15 = 약 67초
```

---

### 궁수 (Ranged)

원거리 공격 유닛입니다. 높은 공격력과 긴 사거리를 가집니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 궁수 | - |
| **비용** | 80 골드 + 10 나무 | 복합 자원 필요 |
| **HP** | 50 | 낮은 체력 |
| **공격력** | 25 | 높은 공격력 |
| **공격 사거리** | 150 픽셀 | 원거리 공격 |
| **이동 속도** | 1.6 | 보통 |
| **공격 쿨다운** | 1초 | - |

#### 특징

- **장점**: 높은 공격력, 원거리 공격, 안전한 딜링
- **단점**: 비싼 비용, 낮은 체력, 나무 필요
- **추천 상황**: 후방 딜링, 검병 지원, 기지 공략

#### 데미지 계산

```
초당 DPS = 공격력 / 쿨다운 = 25 / 1 = 25 DPS
적 기지 파괴 시간 (단독) = 1000 / 25 = 40초
```

---

### 기사 (Knight)

강력한 근접 탱커 유닛입니다. 높은 체력과 공격력을 가지지만 비용이 비쌉니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 기사 | - |
| **비용** | 120 골드 + 20 나무 + 30 돌 | 3가지 자원 필요 |
| **HP** | 250 | 매우 높은 체력 |
| **공격력** | 30 | 높은 공격력 |
| **공격 사거리** | 35 픽셀 | 근접 공격 |
| **이동 속도** | 1.3 | 느린 편 |
| **공격 쿨다운** | 1초 | - |

#### 특징

- **장점**: 높은 체력(탱커), 높은 공격력, 전선 유지
- **단점**: 매우 비싼 비용, 3가지 자원 필요, 느린 생산
- **추천 상황**: 중후반 주력, 전선 탱킹, 기지 공략

#### 데미지 계산

```
초당 DPS = 공격력 / 쿨다운 = 30 / 1 = 30 DPS
적 기지 파괴 시간 (단독) = 1000 / 30 ≈ 33초
체력 효율 = 250 HP (검병의 2.5배)
```

## 지원 유닛

지원 유닛은 자원을 자동으로 채집합니다. 공격받으면 낮은 공격력으로 반격합니다.

| 유닛 | 반격 공격력 | 반격 사거리 |
|------|------------|------------|
| 나무꾼 | 5 | 25 |
| 광부 | 6 | 25 |
| 채집꾼 | 3 | 20 |
| 금광부 | 4 | 25 |

### 나무꾼 (Woodcutter)

나무를 채집하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 나무꾼 | - |
| **비용** | 30 골드 | 가장 저렴 |
| **HP** | 60 | 보통 |
| **채집 속도** | 1.0 | 기준 속도 |
| **이동 속도** | 1.5 | 보통 |
| **채집 자원** | 나무 (wood) | - |

#### 특징

- **장점**: 저렴함, 나무는 궁수/건설에 필수
- **단점**: 전투 불가, 느린 이동
- **추천**: 게임 시작 직후 2~3명 고용

#### 효율 분석

```
채집 효율 = 30 골드로 지속적인 나무 공급
손익분기점 = 약 30초 (나무 30 이상 채집 시)
```

---

### 광부 (Miner)

돌을 채집하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 광부 | - |
| **비용** | 40 골드 + 5 나무 | 나무 필요 |
| **HP** | 70 | 높은 편 |
| **채집 속도** | 0.8 | 느림 |
| **이동 속도** | 1.5 | 보통 |
| **채집 자원** | 돌 (stone) | - |

#### 특징

- **장점**: 높은 체력, 돌은 건설/업그레이드에 필수, 적당한 이동 속도
- **단점**: 느린 채집, 나무 비용 필요
- **추천**: 나무꾼 고용 후 1~2명 고용

#### 효율 분석

```
채집 효율 = 나무꾼보다 20% 느림
필요 시점 = 벽 건설, 기지 업그레이드 전
```

---

### 채집꾼 (Gatherer)

약초를 채집하고, 드물게 수정을 발견하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 채집꾼 | - |
| **비용** | 35 골드 | 저렴 |
| **HP** | 50 | 낮음 |
| **채집 속도** | 1.2 | 가장 빠름 |
| **이동 속도** | 1.5 | 빠름 |
| **채집 자원** | 약초 (herb) | 수정 확률 있음 |

#### 특징

- **장점**: 빠른 채집, 빠른 이동, 수정 획득 가능성
- **단점**: 낮은 체력, 약초 용도 제한적
- **추천**: 여유 자원이 있을 때 고용

#### 수정 획득

```
수정 발견 확률 = 0.1% (약초 채집 시)
희귀하지만 가치 있는 자원
```

---

### 금광부 (Gold Miner)

광산에서 골드를 채집하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 금광부 | - |
| **비용** | 100 골드 + 20 나무 | 가장 비쌈 |
| **HP** | 70 | 보통 |
| **채집 속도** | 1.5 | 가장 빠름 |
| **이동 속도** | 1.5 | 보통 |
| **채집 자원** | 골드 (gold) | 광산에서만 채집 |

#### 특징

- **장점**: 빠른 골드 채집, 자동 수입 증가
- **단점**: 높은 비용, 광산 필요
- **추천**: 중반 이후 자원이 여유로울 때 고용

#### 효율 분석

```
채집 효율 = 1.5 골드/초
손익분기점 = 약 67초 (100 골드 회수)
자동 수입(2/초) 대비 75% 추가 수입
```

## 유닛 상태

유닛은 다음 4가지 상태 중 하나를 가집니다:

| 상태 | 영문 | 설명 |
|------|------|------|
| 대기 | idle | 행동 대기 중 |
| 이동 | moving | 목표 지점으로 이동 중 |
| 공격 | attacking | 적 유닛/기지 공격 중 |
| 채집 | gathering | 자원 채집 중 (지원 유닛만) |

## 유닛 운용 전략

### 전투 유닛 비율

| 상황 | 검병 : 궁수 : 기사 | 이유 |
|------|-------------------|------|
| 초반 | 3 : 0 : 0 | 검병만 생산 가능 |
| 중반 | 2 : 1 : 0 | 궁수 추가 |
| 후반 | 1 : 2 : 1 | 기사 탱커 + 궁수 화력 |

### 지원 유닛 권장 수

| 유닛 | 권장 수 | 이유 |
|------|---------|------|
| 나무꾼 | 2~3명 | 나무 지속 공급 |
| 광부 | 1~2명 | 건설용 돌 확보 |
| 채집꾼 | 0~1명 | 선택적 |

### 비용 효율 비교

| 유닛 | 비용 | 전투력/비용 | 평가 |
|------|------|------------|------|
| 검병 | 50G | 0.30 | 가성비 좋음 |
| 궁수 | 80G + 10W | 0.28 | DPS 높음 |
