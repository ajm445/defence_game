## 유닛 개요

유닛은 크게 **전투 유닛**과 **지원 유닛** 두 가지로 분류됩니다.

| 구분 | 역할 | 유닛 종류 |
|------|------|----------|
| 전투 유닛 | 적 유닛/기지 공격 | 검병, 궁수, 기사, 마법사 |
| 지원 유닛 | 자원 채집 / 아군 회복 | 나무꾼, 광부, 채집꾼, 금광부, 힐러 |

### 공통 속성

| 속성 | 설명 |
|------|------|
| HP | 유닛의 체력, 0이 되면 사망 |
| Speed | 이동 속도 (픽셀/프레임) |
| Cost | 생산에 필요한 자원 |

## 전투 유닛

전투 유닛은 적 유닛을 공격하고, 적 기지를 파괴하는 역할을 합니다.

### 타겟팅 우선순위

전투 유닛은 다음 우선순위로 타겟을 선택합니다:

| 순위 | 타겟 | 조건 |
|------|------|------|
| 1순위 | 적 유닛 | 벽/본진보다 가까운 적 유닛이 있으면 먼저 접근 |
| 2순위 | 적 벽 | 본진보다 가까운 벽이 있으면 공격 |
| 3순위 | 적 본진 | 다른 타겟이 없으면 본진으로 이동 |

> 사거리 밖의 적 유닛도 가까우면 먼저 접근합니다.

---

### 검병 (Melee)

근접 전투 전문 유닛입니다. 저렴한 비용으로 대량 생산이 가능합니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 검병 | - |
| **비용** | 50 골드 | 골드만 필요 |
| **HP** | 100 | 높은 체력 |
| **공격력** | 15 | 기본 공격력 |
| **공격 사거리** | 30 픽셀 | 근접 공격 |
| **이동 속도** | 1.5 | 보통 |
| **공격 쿨다운** | 1초 | - |

#### 특징

- **장점**: 저렴함, 높은 체력, 빠른 이동 속도
- **단점**: 근접 공격만 가능, 궁수에게 취약
- **추천 상황**: 초반 러시, 물량 전투, 탱킹

#### 데미지 계산

```
초당 DPS = 공격력 / 쿨다운 = 15 / 1 = 15 DPS
적 기지 파괴 시간 (단독) = 1000 / 15 = 약 67초
```

---

### 궁수 (Ranged)

원거리 공격 유닛입니다. 높은 공격력과 긴 사거리를 가집니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 궁수 | - |
| **비용** | 80 골드 + 10 나무 | 복합 자원 필요 |
| **HP** | 50 | 낮은 체력 |
| **공격력** | 25 | 높은 공격력 |
| **공격 사거리** | 150 픽셀 | 원거리 공격 |
| **이동 속도** | 1.6 | 보통 |
| **공격 쿨다운** | 1초 | - |

#### 특징

- **장점**: 높은 공격력, 원거리 공격, 안전한 딜링
- **단점**: 비싼 비용, 낮은 체력, 나무 필요
- **추천 상황**: 후방 딜링, 검병 지원, 기지 공략

#### 데미지 계산

```
초당 DPS = 공격력 / 쿨다운 = 25 / 1 = 25 DPS
적 기지 파괴 시간 (단독) = 1000 / 25 = 40초
```

### 기사 (Knight)

강력한 근접 탱커 유닛입니다. 매우 높은 체력으로 전선을 유지하는 역할을 합니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 기사 | - |
| **비용** | 120 골드 + 20 나무 + 30 돌 | 3가지 자원 필요 |
| **HP** | 250 | 매우 높은 체력 |
| **공격력** | 10 | 낮은 공격력 |
| **공격 사거리** | 35 픽셀 | 근접 공격 |
| **이동 속도** | 1.3 | 느린 편 |
| **공격 쿨다운** | 1초 | - |

#### 특징

- **장점**: 매우 높은 체력(탱커), 전선 유지, 적 공격 흡수
- **단점**: 매우 비싼 비용, 3가지 자원 필요, 낮은 공격력
- **추천 상황**: 중후반 탱킹, 딜러 보호, 적 화력 분산

#### 데미지 계산

```
초당 DPS = 공격력 / 쿨다운 = 10 / 1 = 10 DPS
적 기지 파괴 시간 (단독) = 1000 / 10 = 100초
체력 효율 = 250 HP (검병의 2.5배)
```

---

### 마법사 (Mage)

강력한 범위 공격 마법 유닛입니다. 넓은 사거리와 AOE(광역) 공격이 특징입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 마법사 | - |
| **비용** | 150 골드 + 10 수정 | 희귀 자원 필요 |
| **HP** | 40 | 낮은 체력 |
| **공격력** | 50 | 매우 높은 공격력 |
| **공격 사거리** | 180 픽셀 | 매우 긴 사거리 |
| **AOE 반경** | 50 픽셀 | 범위 공격 |
| **이동 속도** | 1.2 | 느린 편 |
| **공격 쿨다운** | 2초 | 긴 쿨다운 |

#### 특징

- **장점**: 범위 공격(AOE), 매우 긴 사거리, 매우 높은 공격력
- **단점**: 낮은 체력, 비싼 비용, 수정 필요, 느린 공격 속도
- **추천 상황**: 밀집된 적 무리 공격, 후방 딜링, 벽 파괴

#### AOE 데미지 계산

```
중심 데미지 = 50 (100%)
가장자리 데미지 = 25 (50%)
데미지 감소 = 중심에서 거리에 비례

초당 DPS = 50 / 2 = 25 DPS (단일 타겟)
다중 타겟 시 DPS 크게 증가
```

#### 수정 획득 방법

- 맵 중앙 수정 노드에서 채집 (3개 한정)
- 채집꾼이 약초 채집 시 0.1% 확률로 획득

## 지원 유닛

지원 유닛은 자원을 자동으로 채집하거나 아군을 회복합니다. 공격받으면 낮은 공격력으로 반격합니다.

| 유닛 | 반격 공격력 | 반격 사거리 |
|------|------------|------------|
| 나무꾼 | 5 | 25 |
| 광부 | 6 | 25 |
| 채집꾼 | 3 | 20 |
| 금광부 | 4 | 25 |
| 힐러 | 3 | 25 |

### 나무꾼 (Woodcutter)

나무를 채집하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 나무꾼 | - |
| **비용** | 30 골드 | 가장 저렴 |
| **HP** | 60 | 보통 |
| **채집 속도** | 1.0 | 기준 속도 |
| **이동 속도** | 1.5 | 보통 |
| **채집 자원** | 나무 (wood) | - |

#### 특징

- **장점**: 저렴함, 나무는 궁수/건설에 필수
- **단점**: 전투 불가, 느린 이동
- **추천**: 게임 시작 직후 2~3명 고용

#### 효율 분석

```
채집 효율 = 30 골드로 지속적인 나무 공급
손익분기점 = 약 30초 (나무 30 이상 채집 시)
```

### 광부 (Miner)

돌을 채집하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 광부 | - |
| **비용** | 40 골드 + 5 나무 | 나무 필요 |
| **HP** | 70 | 높은 편 |
| **채집 속도** | 0.8 | 느림 |
| **이동 속도** | 1.5 | 보통 |
| **채집 자원** | 돌 (stone) | - |

#### 특징

- **장점**: 높은 체력, 돌은 건설/업그레이드에 필수, 적당한 이동 속도
- **단점**: 느린 채집, 나무 비용 필요
- **추천**: 나무꾼 고용 후 1~2명 고용

#### 효율 분석

```
채집 효율 = 나무꾼보다 20% 느림
필요 시점 = 벽 건설, 기지 업그레이드 전
```

### 채집꾼 (Gatherer)

약초를 채집하고, 드물게 수정을 발견하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 채집꾼 | - |
| **비용** | 35 골드 | 저렴 |
| **HP** | 50 | 낮음 |
| **채집 속도** | 1.2 | 가장 빠름 |
| **이동 속도** | 1.5 | 빠름 |
| **채집 자원** | 약초 (herb) | 수정 확률 있음 |

#### 특징

- **장점**: 빠른 채집, 빠른 이동, 수정 획득 가능성
- **단점**: 낮은 체력, 약초 용도 제한적
- **추천**: 여유 자원이 있을 때 고용

#### 수정 획득

```
수정 발견 확률 = 0.1% (약초 채집 시)
희귀하지만 가치 있는 자원
```

### 금광부 (Gold Miner)

광산에서 골드를 채집하는 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 금광부 | - |
| **비용** | 100 골드 + 20 나무 | 가장 비쌈 |
| **HP** | 70 | 보통 |
| **채집 속도** | 1.5 | 가장 빠름 |
| **이동 속도** | 1.5 | 보통 |
| **채집 자원** | 골드 (gold) | 광산에서만 채집 |

#### 특징

- **장점**: 빠른 골드 채집, 자동 수입 증가
- **단점**: 높은 비용, 광산 필요
- **추천**: 중반 이후 자원이 여유로울 때 고용

#### 효율 분석

```
채집 효율 = 1.5 골드/초
손익분기점 = 약 67초 (100 골드 회수)
자동 수입(기본 4/초) 대비 37.5% 추가 수입
```

### 힐러 (Healer)

아군 유닛을 치료하는 지원 유닛입니다.

#### 스탯

| 속성 | 값 | 설명 |
|------|-----|------|
| **이름** | 힐러 | - |
| **비용** | 70 골드 + 15 약초 | 약초 필요 |
| **HP** | 60 | 보통 |
| **회복량** | 10 HP/초 | 지속 회복 |
| **회복 사거리** | 100 픽셀 | 회복 범위 |
| **이동 속도** | 1.4 | 빠른 편 |

#### 특징

- **장점**: 아군 HP 회복, 전투 지속력 증가, 빠른 이동
- **단점**: 약초 필요, 전투력 없음, 단일 대상 치료
- **추천 상황**: 전투 유닛 지원, 장기전 대비

#### 회복 로직

```
1. HP가 가장 낮은 아군 탐색 (HP < maxHP)
2. 해당 아군에게 이동
3. 회복 사거리(100px) 내 도달 시 회복 시작
4. 치료할 대상이 없으면 본진 근처 대기
```

#### 효율 분석

```
회복 효율 = 10 HP/초
검병 풀회복 시간 = 100 / 10 = 10초
기사 풀회복 시간 = 250 / 10 = 25초
```

## 유닛 상태

유닛은 다음 5가지 상태 중 하나를 가집니다:

| 상태 | 영문 | 설명 | 표시 색상 |
|------|------|------|----------|
| 대기 | idle | 행동 대기 중 | 없음 |
| 이동 | moving | 목표 지점으로 이동 중 | 없음 |
| 공격 | attacking | 적 유닛/기지 공격 중 | 빨강 |
| 채집 | gathering | 자원 채집 중 (지원 유닛만) | 초록 |
| 회복 | healing | 아군 치료 중 (힐러만) | 핑크 |

## 유닛 운용 전략

### 전투 유닛 비율

| 상황 | 검병 : 궁수 : 기사 : 마법사 | 이유 |
|------|---------------------------|------|
| 초반 | 3 : 0 : 0 : 0 | 검병만 생산 가능 |
| 중반 | 2 : 1 : 0 : 0 | 궁수 추가 |
| 후반 | 1 : 2 : 1 : 1 | 기사 탱커 + 궁수/마법사 화력 |

### 지원 유닛 권장 수

| 유닛 | 권장 수 | 이유 |
|------|---------|------|
| 나무꾼 | 2~3명 | 나무 지속 공급 |
| 광부 | 1~2명 | 건설용 돌 확보 |
| 채집꾼 | 1~2명 | 약초 확보 (힐러용) |
| 힐러 | 1~2명 | 전투 유닛 지원 (전투 유닛 3명 이상 시) |

### 비용 효율 비교

| 유닛 | 비용 | 전투력/비용 | 평가 |
|------|------|------------|------|
| 검병 | 50G | 0.30 | 가성비 좋음 |
| 궁수 | 80G + 10W | 0.28 | DPS 높음 |
| 기사 | 120G + 20W + 30S | 0.25 | 탱커 역할 |
| 마법사 | 150G + 10수정 | 0.23 (AOE 시 증가) | 범위 공격 |
