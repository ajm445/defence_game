# RPG 모드 게임 설명서 (V1.15.22)

## 게임 소개

RPG 모드는 플레이어가 4가지 직업 중 하나를 선택하여 영웅을 조종하고, 중앙의 넥서스를 수호하는 **넥서스 디펜스** 모드입니다. 양쪽 적 기지에서 몰려오는 적들을 막고, 적 기지를 파괴한 후 보스를 처치하여 승리하세요!

### 승리 조건
1. **양쪽 적 기지 파괴** (좌측/우측)
2. **보스 2마리 처치** (기지 파괴 후 스폰)

### 패배 조건
- 넥서스의 HP가 0이 되면 패배
- 싱글플레이: 영웅 사망 시 패배
- 멀티플레이: 넥서스가 파괴되면 패배 (영웅은 부활 가능)

## 조작법

| 키/입력 | 동작 |
|---------|------|
| **WASD** | 이동 |
| **자동** | 기본 공격 (사거리 내 적 자동 공격) |
| **Shift** | 스킬 (마우스 방향으로 시전) |
| **R** | 궁극기 (마우스 방향으로 시전) |
| **1** | 공격력 업그레이드 |
| **2** | 이동속도 업그레이드 |
| **3** | HP 업그레이드 |
| **4** | 추가 골드 업그레이드 |
| **C (누르고 있기)** | 기본 공격 사거리 표시 |
| **Space** | 카메라를 영웅 위치로 이동 |
| **ESC** | 일시정지 |
| **마우스 휠** | 줌 인/아웃 |

## 직업 소개

### 전사 (Warrior)

**특징**: 균형 잡힌 스탯으로 근접 전투에 특화된 직업입니다.

| 스탯 | 수치 |
|------|------|
| HP | 350 |
| 공격력 | 35 |
| 공격속도 | 1.0초 |
| 이동속도 | 1.8 |
| 사거리 | 80 |

**스킬**:
- **자동 - 강타**: 단일 대상 공격 (쿨타임 1.0초)
- **Shift - 돌진**: 전방으로 돌진하며 경로상 적에게 데미지 (쿨타임 6초)
- **R - 광전사**: 10초간 공격력 50% 증가, 공격속도 30% 증가, 피해흡혈 50% (쿨타임 30초)

---

### 궁수 (Archer)

**특징**: 높은 공격력과 빠른 공격속도를 가진 원거리 딜러입니다. 체력이 낮으므로 거리 유지가 중요합니다.

| 스탯 | 수치 |
|------|------|
| HP | 250 |
| 공격력 | 45 |
| 공격속도 | 0.7초 |
| 이동속도 | 2.2 |
| 사거리 | 150 |

**스킬**:
- **자동 - 속사**: 원거리 단일 공격 (쿨타임 0.7초)
- **Shift - 관통 화살**: 일직선으로 관통하는 화살, 공격력 150% 데미지 (쿨타임 5초)
- **R - 화살 비**: 범위 내 모든 적에게 공격력 200% 데미지 (쿨타임 25초)

---

### 기사 (Knight)

**특징**: 높은 체력과 방어 능력을 가진 탱커입니다. 이동속도가 느리지만 생존력이 뛰어나며, 레벨업 시 다른 직업보다 더 많은 HP를 얻습니다.

| 스탯 | 수치 |
|------|------|
| HP | 450 |
| 공격력 | 30 |
| 공격속도 | 1.3초 |
| 이동속도 | 2.1 |
| 사거리 | 80 |
| 레벨업 HP | +50 (다른 직업: +30) |

**스킬**:
- **자동 - 방패 타격**: 근접 공격 (쿨타임 1.3초), 적 적중 시 Shift 스킬 쿨타임 1초 감소
- **Shift - 방패 돌진**: 전방으로 돌진하며 경로상 적에게 최대 HP 10% 데미지 + 2초 기절 (쿨타임 8초)
- **R - 철벽 방어**: 5초간 받는 데미지 70% 감소, HP 20% 회복 (쿨타임 35초)

---

### 마법사 (Mage)

**특징**: 높은 범위 공격력을 가진 마법 딜러입니다. 공격속도가 느리지만 강력한 광역 피해를 줍니다.

| 스탯 | 수치 |
|------|------|
| HP | 220 |
| 공격력 | 55 |
| 공격속도 | 1.8초 |
| 이동속도 | 1.9 |
| 사거리 | 120 |

**스킬**:
- **자동 - 마법 화살**: 원거리 단일 공격 (쿨타임 1.8초)
- **Shift - 화염구**: 범위 공격, 공격력 180% 데미지 (쿨타임 7초)
- **R - 운석 낙하**: 대범위 공격, 공격력 300% 데미지, 3초 후 발동 (쿨타임 40초)

## 패시브 성장 시스템

캐릭터(직업) 레벨 5부터 직업별 패시브가 활성화됩니다. 패시브 효과는 캐릭터 레벨에 비례하여 강해집니다.

### 패시브 활성화 조건

- **캐릭터 레벨 5 이상**: 패시브 활성화
- 패시브 레벨 = 캐릭터 레벨 - 4 (예: 캐릭터 Lv.10 = 패시브 Lv.6)

### 패시브 종류

| 직업 | 패시브 | 효과 |
|------|--------|------|
| 전사 | 피해흡혈 | 공격 시 데미지의 일정 비율만큼 HP 회복 |
| 궁수 | 다중타겟 | 기본 공격 시 추가 대상 공격 확률 |
| 기사 | HP재생 | 초당 HP 자동 회복 |
| 마법사 | 데미지 보너스 | 모든 스킬 데미지 증가 |

### 패시브 상세 설명

- **전사 - 피해흡혈**: 공격 시 데미지의 일정 비율만큼 HP를 회복합니다. 광전사(E) 버프의 피해흡혈과 **곱연산**으로 적용됩니다.
  - 공식: `(1 + 패시브) × (1 + 버프) - 1`

- **궁수 - 다중타겟**: 기본 공격 시 확률적으로 다중 대상을 공격합니다.

- **기사 - HP 재생**: 초당 HP가 자동으로 회복됩니다.

- **마법사 - 데미지 보너스**: 모든 스킬(기본 공격/스킬/궁극기) 데미지가 증가합니다.

화면 우하단에 미니맵이 표시됩니다.

- **밝은 영역**: 현재 시야 범위 (반경 300픽셀)

- **붉은 점**: 적 위치 (시야 내 적만 표시)
- **흰색 사각형**: 현재 카메라 뷰포트
- **파란색 (중앙)**: 넥서스 위치
- **붉은색 (좌우)**: 적 기지 위치 (파괴 시 회색)

## 넥서스 디펜스 시스템

### 맵 구성

- **맵 크기**: 3000 x 2000 (가로로 넓은 레인 형태)
- **넥서스**: 맵 중앙 (1500, 1000)에 위치한 수호 대상 (HP: 5000)
- **적 기지**: 좌측 (150) / 우측 (2850)에 위치 (각 HP: 3000)
- **레인**: 적 기지에서 넥서스까지 이어지는 경로 (시각적으로 표시됨)
- 적은 양쪽 기지에서 **동시에** 스폰되어 레인을 따라 넥서스로 진격합니다

### 게임 진행

1. **플레이 단계**: 양쪽 적 기지에서 적이 동시 스폰, 넥서스를 향해 진격
2. **기지 파괴**: 영웅이 적 기지를 공격하여 파괴
3. **보스 단계**: 양쪽 기지 파괴 시 보스 2마리 스폰
4. **승리**: 보스 2마리 처치

### 시간 경과에 따른 변화

- **스폰 간격**: 점점 빨라짐 (최소 1.5초)
- **스폰 수량**: 시간에 따라 증가
  - 0~2분: 기지당 1마리
  - 2~4분: 기지당 1~2마리
  - 4~6분: 기지당 2마리
  - 6분 이후: 기지당 2~3마리
- **적 스탯**: 시간이 지날수록 강해짐
- **적 종류**: 시간에 따라 다양한 적 등장

## 골드 & 업그레이드 시스템

### 골드 획득

- 적을 처치하면 골드를 획득합니다
- 추가 골드 업그레이드로 획득량 증가 가능 (레벨당 +2 골드)

| 적 유형 | 기본 골드 |
|---------|-----------|
| 검병 | 12 |
| 궁수 | 18 |
| 기사 | 35 |
| 마법사 | 45 |
| 보스 | 500 |

### 업그레이드

골드를 사용하여 영웅을 강화할 수 있습니다. 최대 레벨은 캐릭터(직업) 레벨입니다.

| 업그레이드 | 키 | 효과 |
|------------|-----|------|
| 공격력 | 1 | 레벨당 +3 공격력 |
| 이동속도 | 2 | 레벨당 +0.08 속도 |
| HP | 3 | 레벨당 +25 최대 HP |
| 추가 골드 | 4 | 레벨당 +2 골드 |

### SP 스탯 업그레이드 (프로필)

프로필에서 SP를 투자하여 영구적으로 캐릭터를 강화할 수 있습니다. 이 보너스는 게임 시작 시 자동 적용됩니다.

| 스탯 | 적용 대상 | 효과 |
|------|-----------|------|
| 공격력 | 모든 직업 | 레벨당 +5 공격력 |
| 이동속도 | 모든 직업 | 레벨당 +0.1 속도 |
| HP | 모든 직업 | 레벨당 +30 최대 HP |
| 사거리 | 궁수, 마법사 | 레벨당 +10 사거리 |
| HP 재생 | 전사, 기사 | 레벨당 +2/초 HP 재생 |

### 업그레이드 비용

- 기본 비용: 50 골드
- 레벨당 비용 증가: 1.5배
- 공식: `50 × 1.5^현재레벨`

## 게임 팁

1. **넥서스 수호**: 넥서스 HP를 항상 확인하세요. 적이 넥서스에 도달하기 전에 처리하세요.

2. **어그로 활용**: 적을 공격하면 5초간 어그로가 끌립니다. 넥서스에서 멀리 유인하세요.

3. **기지 파괴 타이밍**: 골드를 충분히 모아 업그레이드한 후 기지를 공격하세요.

4. **업그레이드 우선순위**: 초반에는 공격력/골드 획득률, 후반에는 HP/속도 업그레이드를 권장합니다.

5. **전사**: 광전사 버프를 활성화하면 피해흡혈로 체력을 회복하며 싸울 수 있습니다.

6. **궁수/마법사**: 거리를 유지하며 공격하세요. 적이 다가오면 즉시 후퇴하세요.

7. **기사**: 철벽 방어를 적절히 사용하여 위험한 상황에서 생존하세요. 넥서스 근처에서 탱킹하세요.

8. **보스전 준비**: 양쪽 기지 파괴 전에 궁극기 쿨타임과 충분한 업그레이드를 확인하세요.

9. **멀티플레이**: 역할 분담이 중요합니다. 탱커는 넥서스 수호, 딜러는 기지 공격을 담당하세요.

## 버전 히스토리

### V1.15.22
- **멀티플레이어 기본 공격 AoE 버그 수정**
  - 클라이언트 전사/기사/마법사 기본 공격이 단일 타겟으로 적용되던 문제 해결
  - 호스트와 동일하게 범위 공격(전방 각도 기반) 적용
- **멀티플레이어 AoE 공격 시 적 기지 데미지 적용**
  - 범위 공격 시 적과 기지가 함께 있을 때 기지에도 데미지 적용
  - 기존: 적만 공격되고 기지는 공격되지 않음
- **광전사 버프 피해흡혈 모든 클래스 적용**
  - 광전사 버프를 공유받은 모든 클래스에 피해흡혈 효과 적용
  - 전사 패시브 피해흡혈과 버프 피해흡혈 분리 계산: `(1 + 패시브) × (1 + 버프) - 1`
- **클라이언트 이동 러버밴딩 수정**
  - 클라이언트 캐릭터가 이동 중 반대 방향으로 당겨지던 현상 수정
  - 이동 중에는 로컬 위치 우선, 정지 시에만 서버 위치 보정
