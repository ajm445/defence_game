# RPG 모드 게임 설명서 (V1.22.9)

## 게임 소개

RPG 모드는 플레이어가 4가지 직업 중 하나를 선택하여 영웅을 조종하고, 중앙의 넥서스를 수호하는 **넥서스 디펜스** 모드입니다. 적 기지에서 몰려오는 적들을 막고, 적 기지를 파괴한 후 보스를 처치하여 승리하세요!

### 승리 조건
1. **모든 적 기지 파괴**
   - 싱글/2인: 좌측, 우측 (2개)
   - 3인: 좌측, 우측, 상단 (3개)
   - 4인: 좌측, 우측, 상단, 하단 (4개)
2. **보스 처치** (파괴한 기지 수만큼 스폰)

### 패배 조건
- 넥서스의 HP가 0이 되면 패배
- 싱글플레이: 영웅 사망해도 게임 계속 진행 (부활 후 전투 재개)
- 멀티플레이: 넥서스가 파괴되면 패배 (영웅은 부활 가능)

## 조작법

| 키/입력 | 동작 |
|---------|------|
| **WASD** | 이동 |
| **자동** | 기본 공격 (사거리 내 적 자동 공격) |
| **Shift** | 스킬 (마우스 방향으로 시전) |
| **R** | 궁극기 (마우스 방향으로 시전) |
| **1** | 공격력 업그레이드 |
| **2** | 이동속도 업그레이드 |
| **3** | HP 업그레이드 |
| **4** | 공격속도 업그레이드 |
| **5** | 골드 획득률 업그레이드 |
| **6** | 사거리 업그레이드 (궁수/마법사) |
| **C (누르고 있기)** | 기본 공격 사거리 표시 |

| **ESC** | 일시정지 |

## 직업 소개

### 전사 (Warrior)

**특징**: 균형 잡힌 스탯으로 근접 전투에 특화된 직업입니다.

| 스탯 | 수치 |
|------|------|
| HP | 350 |
| 공격력 | 35 |
| 공격속도 | 1.0초 |
| 이동속도 | 1.8 |
| 사거리 | 80 |

**스킬**:
- **자동 - 강타**: 단일 대상 공격 (쿨타임 1.0초)
- **Shift - 돌진**: 전방으로 돌진하며 경로상 적에게 데미지 (쿨타임 6초)
- **R - 광전사**: 10초간 공격력 50% 증가, 공격속도 30% 증가, 피해흡혈 50% (쿨타임 30초)

---

### 궁수 (Archer)

**특징**: 높은 공격력과 빠른 공격속도를 가진 원거리 딜러입니다. 체력이 낮으므로 거리 유지가 중요합니다.

| 스탯 | 수치 |
|------|------|
| HP | 250 |
| 공격력 | 45 |
| 공격속도 | 0.7초 |
| 이동속도 | 2.2 |
| 사거리 | 150 |

**스킬**:
- **자동 - 속사**: 원거리 단일 공격 (쿨타임 0.7초)
- **Shift - 관통 화살**: 일직선으로 관통하는 화살, 공격력 150% 데미지 (쿨타임 5초)
- **R - 화살 비**: 범위 내 모든 적에게 공격력 200% 데미지 (쿨타임 25초)

---

### 기사 (Knight)

**특징**: 높은 체력과 방어 능력을 가진 탱커입니다. 이동속도가 느리지만 생존력이 뛰어나며, 레벨업 시 다른 직업보다 더 많은 HP를 얻습니다.

| 스탯 | 수치 |
|------|------|
| HP | 450 |
| 공격력 | 30 |
| 공격속도 | 1.1초 |
| 이동속도 | 2.1 |
| 사거리 | 80 |
| 레벨업 HP | +50 (다른 직업: +30) |

**스킬**:
- **자동 - 방패 타격**: 근접 공격 (쿨타임 1.1초), 적 적중 시 Shift 스킬 쿨타임 1초 감소
- **Shift - 방패 돌진**: 전방으로 돌진하며 경로상 적에게 최대 HP 10% 데미지 + 2초 기절 (쿨타임 8초)
- **R - 철벽 방어**: 5초간 받는 데미지 70% 감소, HP 20% 회복 (쿨타임 35초)

---

### 마법사 (Mage)

**특징**: 높은 범위 공격력을 가진 마법 딜러입니다. 공격속도가 느리지만 강력한 광역 피해를 줍니다.

| 스탯 | 수치 |
|------|------|
| HP | 220 |
| 공격력 | 55 |
| 공격속도 | 1.8초 |
| 이동속도 | 1.9 |
| 사거리 | 120 |

**스킬**:
- **자동 - 마법 화살**: 원거리 단일 공격 (쿨타임 1.8초)
- **Shift - 화염구**: 범위 공격, 공격력 180% 데미지 (쿨타임 7초)
- **R - 운석 낙하**: 대범위 공격, 공격력 300% 데미지, 3초 후 발동 (쿨타임 40초)

## 패시브 성장 시스템

캐릭터(직업) 레벨 5부터 직업별 패시브가 활성화됩니다. 패시브 효과는 캐릭터 레벨에 비례하여 강해집니다.

### 패시브 활성화 조건

- **캐릭터 레벨 5 이상**: 패시브 활성화
- 패시브 레벨 = 캐릭터 레벨 - 4 (예: 캐릭터 Lv.10 = 패시브 Lv.6)

### 패시브 종류

| 직업 | 패시브 | 효과 |
|------|--------|------|
| 전사 | 피해흡혈 | 공격 시 데미지의 일정 비율만큼 HP 회복 |
| 궁수 | 다중타겟 | 기본 공격 시 추가 대상 공격 확률 |
| 기사 | HP재생 | 초당 HP 자동 회복 |
| 마법사 | 데미지 보너스 | 모든 스킬 데미지 증가 |

### 패시브 상세 설명

- **전사 - 피해흡혈**: 공격 시 데미지의 일정 비율만큼 HP를 회복합니다. 광전사(E) 버프의 피해흡혈과 **곱연산**으로 적용됩니다.
  - 공식: `(1 + 패시브) × (1 + 버프) - 1`

- **궁수 - 다중타겟**: 기본 공격 시 확률적으로 다중 대상을 공격합니다.

- **기사 - HP 재생**: 초당 HP가 자동으로 회복됩니다.

- **마법사 - 데미지 보너스**: 모든 스킬(기본 공격/스킬/궁극기) 데미지가 증가합니다.

### 캐릭터 정보 UI

캐릭터 정보 화면에서 다음 효과들을 확인할 수 있습니다:

- **패시브 효과**: 직업별 패시브와 현재 레벨에 따른 수치
- **전직 특수 효과**: 전직 시 부여되는 고유 특수 효과 (전직 캐릭터만 표시)

| 전직 | 특수 효과 |
|------|-----------|
| 버서커 | 피해흡혈 1.5배 |
| 가디언 | 받는 피해 30% 감소 |
| 저격수 | 크리티컬 확률 50% |
| 레인저 | 다중 타겟 5명 |
| 팔라딘 | 기본공격 시 주변 아군 힐 (본인 제외) |
| 다크나이트 | 피해흡혈 20% 부여 |
| 대마법사 | 보스에게 50% 추가 데미지 |
| 힐러 | 오로라 범위 내 아군 초당 회복 (본인 포함) |

## 전직 시스템

캐릭터 레벨 15에 도달하면 상위 직업으로 전직할 수 있습니다. 각 기본 직업은 2개의 전직 경로 중 하나를 선택할 수 있습니다.

### 전직 조건

| 단계 | 요구 레벨 | 설명 |
|------|----------|------|
| 1차 전직 | 15 | 기본 직업에서 전직 직업으로 변경 |
| 2차 강화 | 40 | 전직 직업의 스탯 20% 증가 (자동 적용) |

### 전직 방법

1. **프로필 UI 접근**: 프로필을 클릭하여 캐릭터 업그레이드 UI를 엽니다.
2. **전직 버튼 활성화**: 레벨 15에 도달하면 전직 버튼이 나타납니다.
3. **전직 선택**: 2가지 전직 경로 중 하나를 선택합니다.
4. **전직 완료**: 외형, 스탯, 스킬이 전직 직업에 맞게 변경됩니다.

> **주의**: 전직 변경 시 레벨 15, SP 14로 초기화되고 스탯 업그레이드가 리셋됩니다.

### 전직 시 변경 사항

**유지되는 것:**
- SP 업그레이드 보너스
- 클래스 레벨 및 경험치

**변경되는 것:**
- 외형: 전직 직업의 새로운 외형
- 기본 스탯: 전직 직업의 향상된 스탯
- W 스킬 (Shift): 전직 직업의 새로운 스킬
- R 스킬: 전직 직업의 새로운 궁극기
- 특수 효과: 전직 직업 고유 효과

### 전직 경로

| 기본 직업 | 전직 1 | 전직 2 |
|-----------|--------|--------|
| 전사 | 버서커 (공격 특화) | 가디언 (방어 특화) |
| 궁수 | 저격수 (단일 공격력) | 레인저 (공격속도) |
| 기사 | 팔라딘 (아군 치유) | 다크나이트 (암흑 공격) |
| 마법사 | 대마법사 (범위 마법) | 힐러 (아군 치유) |

### 전직 직업 상세

#### 버서커 (Berserker) 🔥
- **컨셉**: 공격력과 공격속도 특화 광전사
- **특수 효과**: 피해흡혈 1.5배
- **W - 피의 돌진**: 전방 돌진 + 피해량의 50% 체력 회복 (쿨타임 6초)
- **R - 광란**: 10초간 공격력/공속 100% 증가, 받는 피해 50% 증가 (쿨타임 45초)

#### 가디언 (Guardian) 🛡️
- **컨셉**: 높은 방어력과 체력의 수호자
- **특수 효과**: 받는 피해 30% 감소
- **W - 수호의 돌진**: 전방 돌진 + 적에게 최대 HP 10% 데미지 + 2초 기절 + 보호막 (쿨타임 8초) [기본공격 적중 시 쿨타임 1초 감소]
- **R - 보호막**: 아군 전체 5초간 받는 피해 50% 감소 (쿨타임 40초)

#### 저격수 (Sniper) 🎯
- **컨셉**: 높은 단일 공격력과 최장 사거리
- **특수 효과**: 크리티컬 확률 50%
- **W - 후방 도약**: 뒤로 점프 + 200% 데미지 화살 + 3초간 이동속도 30% 증가 (쿨타임 5초)
- **R - 저격**: 3초 조준 후 1000% 데미지 단일 타격 (쿨타임 30초)

#### 레인저 (Ranger) 🏹
- **컨셉**: 빠른 공격속도와 다중 타겟
- **특수 효과**: 다중 타겟 5명
- **W - 다중 화살**: 부채꼴로 5발 관통 화살 발사 (쿨타임 5초)
- **R - 화살 폭풍**: 6초간 공격 속도 2배 (쿨타임 35초)

#### 팔라딘 (Paladin) ⚜️
- **컨셉**: 신성한 힘으로 아군을 치유하는 성기사
- **특수 효과**: 기본공격 시 주변 200px 내 아군 최대HP 5% 회복 (본인 제외)
- **W - 신성한 돌진**: 전방 돌진 + 적에게 기절 + 주변 아군 HP 10% 회복 (쿨타임 8초) [기본공격 적중 시 쿨타임 1초 감소]
- **R - 신성한 빛**: 아군 전체 HP 30% 회복 + 3초 무적 (쿨타임 60초)

#### 다크나이트 (Dark Knight) ⚔️
- **컨셉**: HP를 소모하여 강한 데미지를 내는 암흑기사
- **특수 효과**: 피해흡혈 20% 부여
- **W - 암흑 찌르기**: 1초 시전 후 전방 직선 150px×80px 범위에 350% 데미지, HP 20% 소모 (쿨타임 4초)
- **R - 어둠의 칼날**: 토글 온/오프, 초당 HP 5% 소모, 주변 150px 내 적에게 초당 공격력 120% 데미지, HP ≤10% 또는 기절 시 자동 해제 (재사용 딜레이 2초)

#### 대마법사 (Archmage) 🌟
- **컨셉**: 강력한 범위 마법 딜러
- **특수 효과**: 보스에게 50% 추가 데미지
- **W - 폭발 화염구**: 대형 화염구 250% 데미지 + 3초 화상 (초당 20% 데미지) (쿨타임 7초)
- **R - 메테오 샤워**: 5초간 운석 10개 낙하 (각 300% 데미지) (쿨타임 50초)

#### 힐러 (Healer) 💚
- **컨셉**: 아군을 치유하는 서포터
- **특수 효과**: 주변 150px 오로라 범위 내 아군 초당 최대HP 4% 회복 (본인 포함)
- **W - 치유의 빛**: 적에게 100% 데미지 + 범위 내 아군 HP 15% 회복 (쿨타임 7초)
- **R - 생명의 샘**: 10초간 아군 전체 초당 최대 HP의 10% 회복 (쿨타임 45초)

| 적 유형 | 공격 사거리 | 이동속도 | 기본 공격력 |

| 검병 | 60 | 1.5 | 15 |

| 기사 | 70 | 1.3 | 12 |

### 적 AI 행동

- **기본 행동**: 넥서스를 향해 이동하며 공격
- **어그로 시스템**: 영웅에게 공격받으면 5초간 해당 영웅을 추적
- **어그로 만료**: 5초 후 다시 넥서스를 향해 이동
- **시간 경과**: 게임 시간이 지날수록 적의 스탯이 증가

## 미니맵 시스템

화면 우하단에 미니맵이 표시됩니다.

- **밝은 영역**: 현재 시야 범위 (반경 300픽셀)
- **흐린 영역**: 이전에 탐사한 영역
- **검은 영역**: 아직 방문하지 않은 영역
- **금색 점**: 영웅 위치
- **붉은 점**: 적 위치 (시야 내 적만 표시)

- **파란색 (중앙)**: 넥서스 위치
- **붉은색 (좌우)**: 적 기지 위치 (파괴 시 회색)

## 넥서스 디펜스 시스템

### 맵 구성

- **맵 크기**: 3000 x 2000 (가로로 넓은 레인 형태)
- **넥서스**: 맵 중앙 (1500, 1000)에 위치한 수호 대상 (HP: 5000)
- **적 기지 위치** (각 HP: 3000):
  - **좌측**: (150, 1000) - 항상 활성화
  - **우측**: (2850, 1000) - 항상 활성화
  - **상단**: (1500, 150) - 3인 이상 멀티플레이 시 활성화
  - **하단**: (1500, 1850) - 4인 멀티플레이 시 활성화
- **레인**: 적 기지에서 넥서스까지 이어지는 경로 (시각적으로 표시됨)
- 적은 활성화된 모든 기지에서 **동시에** 스폰되어 레인을 따라 넥서스로 진격합니다

### 플레이어 수별 기지 구성

| 플레이어 수 | 활성화 기지 | 보스 수 |
|------------|------------|--------|
| 1명 (싱글) | 좌측, 우측 | 2마리 |
| 2명 | 좌측, 우측 | 2마리 |
| 3명 | 좌측, 우측, 상단 | 3마리 |
| 4명 | 좌측, 우측, 상단, 하단 | 4마리 |

### 게임 진행

1. **플레이 단계**: 활성화된 모든 기지에서 적이 동시 스폰, 넥서스를 향해 진격
2. **기지 파괴**: 영웅이 적 기지를 공격하여 파괴
3. **보스 단계**: 모든 기지 파괴 시 파괴된 기지 수만큼 보스 스폰
4. **승리**: 모든 보스 처치

### 시간 경과에 따른 변화

- **스폰 간격**: 점점 빨라짐 (최소 1.5초)
- **스폰 수량**: 시간에 따라 증가
  - 0~2분: 기지당 1마리
  - 2~4분: 기지당 1~2마리
  - 4~6분: 기지당 2마리
  - 6분 이후: 기지당 2~3마리
- **적 스탯**: 시간이 지날수록 강해짐
- **적 종류**: 시간에 따라 다양한 적 등장
  - 0~2분: 검병만
  - 2~4분: 검병 + 궁수
  - 4~6분: 검병 + 궁수 + 기사
  - 6분 이후: 전 유닛 (검병 + 궁수 + 기사 + 마법사)
- **유닛 등장 시간 가속 (V1.22.7+)**: 높은 난이도일수록 강한 적이 빨리 등장
  - 쉬움: 1.0배 (기본), 중간: 1.3배, 어려움: 1.6배, 극한: 2.0배, 지옥: 2.2배, 종말: 2.7배
  - 예: 극한 난이도에서는 1분에 궁수, 2분에 기사, 3분에 전 유닛 등장

### 업그레이드

골드를 사용하여 영웅을 강화할 수 있습니다. 최대 레벨은 캐릭터(직업) 레벨입니다.

| 업그레이드 | 키 | 효과 |
|------------|-----|------|
| 공격력 | 1 | 레벨당 +3 공격력 |
| 이동속도 | 2 | 레벨당 +0.08 속도 |
| HP | 3 | 레벨당 +25 최대 HP |
| 공격속도 | 4 | 레벨당 -0.05초 공격 쿨다운 |
| 골드획득률 | 5 | 레벨당 +0.5 초당 골드 |
| 사거리 | 6 | 레벨당 +8 사거리 (궁수/마법사 전용) |

### SP 스탯 업그레이드 (프로필)

프로필에서 SP를 투자하여 영구적으로 캐릭터를 강화할 수 있습니다. 이 보너스는 게임 시작 시 자동 적용됩니다.

| 스탯 | 적용 대상 | 효과 |
|------|-----------|------|
| 공격력 | 모든 직업 | 레벨당 +5 공격력 |
| 이동속도 | 모든 직업 | 레벨당 +0.1 속도 |
| HP | 모든 직업 | 레벨당 +30 최대 HP |
| 공격속도 | 모든 직업 | 레벨당 -0.05초 공격 쿨다운 |
| 사거리 | 궁수, 마법사 | 레벨당 +10 사거리 |
| HP 재생 | 전사, 기사 | 레벨당 +2/초 HP 재생 |

### 업그레이드 비용

- **선형 비례 증가 방식**
- 기본 비용: 50 골드
- 공식: `50 × (현재레벨 + 1)`
- 예시: 0→1레벨 50G, 1→2레벨 100G, 2→3레벨 150G...

## VIP 시스템

VIP 플레이어는 게임 종료 시 경험치 보너스를 받습니다.

### VIP 혜택

| 혜택 | 설명 |
|------|------|

| 직업 경험치 2배 | 게임 종료 시 직업(클래스) 경험치 100% 추가 획득 |

- **싱글플레이**: 적용됨

### VIP 획득 방법

VIP 역할은 관리자에 의해 부여됩니다. 일반 플레이어가 직접 VIP를 구매하거나 획득하는 시스템은 현재 없습니다.

## 게임 팁

1. **넥서스 수호**: 넥서스 HP를 항상 확인하세요. 적이 넥서스에 도달하기 전에 처리하세요.

2. **어그로 활용**: 적을 공격하면 5초간 어그로가 끌립니다. 넥서스에서 멀리 유인하세요.

3. **기지 파괴 타이밍**: 골드를 충분히 모아 업그레이드한 후 기지를 공격하세요.

4. **업그레이드 우선순위**: 초반에는 공격력/골드 획득률, 후반에는 HP/속도 업그레이드를 권장합니다.

5. **전사**: 광전사 버프를 활성화하면 피해흡혈로 체력을 회복하며 싸울 수 있습니다.

6. **궁수/마법사**: 거리를 유지하며 공격하세요. 적이 다가오면 즉시 후퇴하세요.

7. **기사**: 철벽 방어를 적절히 사용하여 위험한 상황에서 생존하세요. 넥서스 근처에서 탱킹하세요.

8. **보스전 준비**: 양쪽 기지 파괴 전에 궁극기 쿨타임과 충분한 업그레이드를 확인하세요.

9. **멀티플레이**: 역할 분담이 중요합니다. 탱커는 넥서스 수호, 딜러는 기지 공격을 담당하세요.
